# Unlimited Tools and Skills - No Emblem Cost

This mod removes the Spirit Emblem cost for all Prosthetic Tools and Combat Arts in Sekiro: Shadows Die Twice.  
You can now enjoy experimenting with all tools and skills without any limitations!

---

## Installation
1. Install Sekiro Mod Engine:
   Download it from: https://www.nexusmods.com/sekiro/mods/6
2. Extract the downloaded ZIP file for this mod.
3. Copy the "mods" folder to your Sekiro game directory:
   Example: `C:\Program Files (x86)\Steam\steamapps\common\Sekiro\`
4. Start the game and enjoy!

---

## Features
- Removes Spirit Emblem cost for all Prosthetic Tools and Combat Arts.
- Compatible with most mods (as long as they don't overwrite `gameparam.parambnd.dcx`).

---

## Compatibility
- The mod changes the file `gameparam.parambnd.dcx`.  
- Ensure no other mods overwrite this file for best results.
